# What Is SecureStore?

SecureStore is a desktop security workspace for passwords, secure notes, and encrypted files.

It is designed to feel like a real local-first product rather than a demo:

- branded product shell
- local account system
- onboarding flow
- master-password vault protection
- cyber-security inspired UI
- searchable, organized encrypted records
- SecureStore+ preview mode for future premium features

## SecureStore+

SecureStore+ is a premium-mode preview available from the lock screen. It changes the branding to SecureStore+, adds premium visual treatment, and reserves interface space for secure sharing, encrypted cloud sync, and advanced encryption tooling.
